Help & Capabilities

Chat — Type your message and press Enter or click Send. Attach files with 📎 or drop them. Use @nyx, @morrigan, etc. to address a specific aspect.

Voices — Click an aspect in the sidebar to switch: Morrigan (engineer), Nyx (researcher), Echo (companion), Eris (chaos), Cassandra (oracle), Lilith (sovereign). Lock with 🔓 to prevent auto-routing.

Permissions — Enable Allow Write / Allow Run when you want Layla to modify files or run commands. She will request approval for each action.

Approvals — When Layla needs to write, run code, or run shell commands, she queues an approval. Use this tab to approve or reject.

Study Plans — Add topics Layla will study over time. She advances them when you're active.

Memory — Search her long-term memory (learnings, preferences, facts).

Research — Use "Research repo" in the sidebar to run a full research mission on your workspace. View results here.

Settings — Click the gear icon to change model, paths, and other options.

Shortcuts

  • Ctrl+K — Clear input
  • Ctrl+R — Retry last message
  • Ctrl+/ — Open Help
  • Ctrl+F — Search in chat
  • Escape — Stop recording / close overlays

Troubleshooting

  • Model not loading — Check path in Settings, VRAM, n_gpu_layers. See MODELS.md.
  • Approvals not working — Enable Allow Write / Allow Run in sidebar.
  • Voice not working — pip install sounddevice (CLI) or check browser mic permissions (UI).
