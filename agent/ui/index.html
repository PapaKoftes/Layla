Shortcuts

  • Enter — Send · Shift+Enter — New line
  • Ctrl+K — Clear input
  • Ctrl+R — Regenerate last reply
  • Ctrl+/ — Open Help
  • Ctrl+F — Search in chat
  • Arrow Up / Down — Cycle recent prompts (when input focused)
  • Escape — Stop recording / close overlays

Session backup: exports the active conversation (if any), pending approvals, and a short in-memory history tail.

Message edit / branch — Use Regenerate (Ctrl+R) or edit your text and resend; full branching UX is on the backlog.

Troubleshooting

  • Model not loading — Settings, VRAM, n_gpu_layers. MODELS.md.
  • Health stuck — Server must be up; open Status (or reload). Tabs use CSS only (no hidden attribute) so panels are not blank while data loads.
  • Voice — pip install sounddevice or browser mic permissions.
