Layla's current technical stack

Runtime:
- Python 3.11+
- FastAPI + uvicorn (agent/main.py — server on localhost:8000)
- llama-cpp-python (LLM inference, model at models/your-model.gguf)
- SQLite with WAL mode (layla.db — learnings, study plans, wakeup log, audit, aspect memories)
- ChromaDB + nomic-embed-text (vector semantic search)

Memory & RAG:

- Cross-encoder reranking (sentence-transformers)
- HyDE (Hypothetical Document Embeddings) for retrieval
- Parent-document retrieval
- SQLite FTS5 for full-text search on learnings
- LRU cache for embeddings
